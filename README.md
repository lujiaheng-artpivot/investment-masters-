# Investment Masters Skill

> 投资大师智慧融合引擎 —— 巴菲特、芒格、塔勒布、博格尔、霍华德·马克斯等 14 位投资大师思维体系的 Claude Skill

在 AI 泛滥、信息过载、情绪主导的金融市场里,大多数人亏钱不是因为缺信息,而是因为脑子里装的不是投资者的操作系统。

这个 skill 把 14 位古今投资大师一生验证过的心法,压缩成一套可以在对话中调用的认知框架 —— 不荐股、不预测、不贩卖焦虑,只做一件事:把你脑子里那个追涨杀跌的散户,替换成一个能和复利做朋友的人。

## 它包含什么

**六大基石**(大师们的底层共识):复利的不可中断性、能力圈、市场先生、安全边际、随机性与运气、指数哲学

**14 位投资大师**(按流派组织):

- **价值投资派**:格雷厄姆、巴菲特、芒格、彼得·林奇、费雪、卡拉曼、段永平
- **周期/逆向派**:霍华德·马克斯、邓普顿
- **宏观/对冲派**:达里奥、索罗斯
- **量化/指数派**:博格尔、马尔基尔
- **风险/黑天鹅派**:塔勒布

**五个决策框架**:投资前三问 / 三层金字塔配置 / 芒格逆向清单 / 卖出决策规则 / 霍华德市场体温计

**12 个常见陷阱**:每个都配大师原话驳斥("这次不一样""跌了不能割""闲钱炒股无所谓""加杠杆扩大收益"……)

**7 条绝对不做**:不荐股、不预测、不鼓励杠杆、不承诺收益……

**分层应对策略**:针对新手 / 老韭菜 / 10 年老手 / 情绪化状态四种不同用户情境的差异化建议

**分级书单**:入门 → 进阶 → 高阶,含中文原生推荐(段永平、但斌、唐朝)

## 设计理念

这个 skill 有意做了几个反直觉的选择:

1. **拒绝荐股和预测**。这是大师们的共同原则,skill 本身就不允许越界。
2. **先问情境再给建议**。避免千篇一律的"定投指数"式回答。
3. **情绪优先于框架**。用户在贪婪/恐慌状态下直接讲道理没用,要先降温。
4. **分层建议而不是非黑即白**。每条建议都带条件,让用户自己对号入座。

## 如何使用

### 在 Claude Code / Cowork 中

把 `skills/investment-masters/` 文件夹复制到你的 skills 目录:

```bash
cp -r skills/investment-masters ~/.claude/skills/
```

之后只要对话里涉及投资、理财、股票、基金、某位大师名字,skill 会自动触发。

### 手动调用

在对话里说"用 investment-masters skill 帮我看看……",或者直接粘贴 `SKILL.md` 的内容作为系统提示词使用。

## 触发场景示例

- "手头有 10 万闲钱该怎么投"
- "最近想炒股,从哪开始"
- "朋友推荐了一只股,能买吗"
- "我的基金跌了 30% 要不要赎回"
- "现在是牛市顶还是底"
- "巴菲特的护城河到底是什么意思"
- "要不要加杠杆扩大收益"
- "定投指数基金真的靠谱吗"

## 不适用场景

- 技术分析(K 线、MACD、量能指标)
- 具体股票的买入推荐
- 短期涨跌预测
- 加密货币项目方分析
- 需要专业资质的税务/遗产规划

以上这些要么超出本 skill 能力范围,要么违背大师们的核心哲学。

## 书单(skill 中完整列出,这里只列前三本必读)

1. 约翰·博格尔《共同基金常识》—— 搞清楚为什么指数基金是圣杯
2. 伯顿·马尔基尔《漫步华尔街》—— 搞清楚市场本质
3. 本杰明·格雷厄姆《聪明的投资者》—— 搞清楚价值投资的骨架
